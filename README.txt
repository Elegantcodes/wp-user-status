There are a number of filters to modify the functionality and output of the users including:
* `cf_user_activity_idle_time` - Set the amount of time a user has to be idle before registering as offline
* `cf_user_status_query_args` - Query args for which users to display
* `cf_user_status_users` - List of users which will be displayed
* `cf_user_activity_output_before` - Output before list of users
* `cf_user_activity_output_row` - Output for a single user
* `cf_user_activity_output_after` - Output after list of users
